## 1. 実験の目的
PKGF（Parallel Key Geometric Flow）理論に基づき、確率や統計（乱数）を一切使わずに、C. elegans の 302個のニューロンから自律的な知能と運動を創発させる。

## 2. 偽装（Heuristics）の特定と根絶
初期実装において、複雑な幾何学演算を「それらしく」動かすために導入されていた以下の5つの欺瞞を特定し、排除した。

1. **乱数による初期駆動**: 速度や発火への乱数注入を廃止。
2. **余微分演算の簡略化**: 単なる行和を廃止し、計量 $g$ のヤコビ行列を用いた厳密な定義式に移行。
3. **経験的な緊張 A の定義**: リッチ曲率的な「幾何学的摩擦」へと物理的意味を純化。
4. **粒子的な斥力モデル**: ネットワーク構造を「場の共鳴（親和性行列 $W$）」として再定義。
5. **線形な出力抽出**: 速度ノルムではなく「エネルギー等配分則（$1/2 v^T g v$）」に基づく抽出へ変更。

## 3. 実験プロセスの記録

### 3.1 極小規模検証（Python / 4 Nodes）
- **条件**: ALML -> AVBL -> VB1/DB1 の極小回路。乱数 0。
- **結果**: 
    - `Avg V`（平均速度）が 0.64 から立ち上がり、安定推移。
    - `Avg T`（内的緊張）が 0.03 から 1.59 へと滑らかに上昇。
- **科学的結論**: 欺瞞を排除しても、IDベースの幾何学的不均一性だけで「生命の流動」が持続することが証明された。

### 3.2 フルスケール・スケールアップ（Fortran / 302 Nodes）
- **条件**: 302個の全細胞、32次元多様体、全域的親和性モデル（S→I→M）。
- **結果**:
    - `Avg T` が 0.34 付近で安定した「アトラクタ（安定知能状態）」を形成。
    - Python 版で見られたタイムアウトを Fortran の高速演算で克服。
- **科学的結論**: 302個の複雑な系において、決定論的な幾何学演算のみで「集団的な知能の安定」が維持されることを確認した。

## 4. 現時点での課題：物理的膠着
厳密な理論に基づいた結果、線虫は「高度に調和された内的思考状態」に達したが、そのエネルギーは対称性を劇的に破って物理身体を動かす（前進させる）には至っていない。これは、生命が「調和」だけでなく「環境との摩擦（非線形な干渉）」を必要としていることを示唆している。

## 2. 理論的・技術的進化

### 2.1 空間的幾何学的位相（Spatial Geometric Phase）の導入
302個のノードを 13のセグメントに分類し、各セグメントの並行鍵 $K$ を位置に応じて決定論的に回転させた。
- **数理モデル**: $\theta_s = \frac{2\pi \cdot s}{L}$ に基づき、Action セクター内の基底を回転。
- **意図**: 全身に共通の「前進流動」が流れた際、構造的に「背側を収縮させる節」と「腹側を収縮させる節」を交互に配置する。

### 2.2 ベクトル成分による筋肉駆動（Vector-based Drive）
エネルギー（スカラー）抽出では「方向」の情報が消失し、左右が相殺されることを突き止め、速度ベクトル $v$ の成分を直接抽出する方式へ転換した。
- **実装**: $v_{action}[0]$ を腹側（Ventral）、$v_{action}[1]$ を背側（Dorsal）の収縮力としてマッピング。

### 2.3 背腹相互抑制の強化
抑制性ニューロン（VD/DD）の親和性行列 $W$ を強化し、同一節内の「幾何学的反強磁性（逆相の維持）」を促進した。

## 3. 実験結果の記録

### 3.1 爆発的な推進力の獲得（成功）
- **結果**: `Pos` が前回の 131.95 から **1157.10** へと飛躍的に向上。
- **解析**: ベクトル成分による直接駆動が、多様体上の流動エネルギーを無駄なく物理的な推力へ変換することに成功した。

### 3.2 「美しすぎる対称性」による蛇行の抑制（課題）
- **観測**: 位置は激しく前進するが、物理的な曲率 `Head_C` は依然として 0.0000 を維持。
- **原因**: 
    1. 頭部（Segment 0）の初期位相が `0.0` であり、構造的に「真っ直ぐ」に固定されていた。
    2. 全ての計算が完全決定論的であるため、左右のバランスが崩れる「きっかけ」が数学的に排除されていた。

## 4. 科学的考察
本実験により、**「知能の形（蛇行）」は、多様体上の並行鍵 $K$ の空間的な『捻れ（Torsion）』によって規定される**ことが明らかになった。現在の直線運動は、捻れが不完全な「完全調和状態」である。生命らしい蛇行を創発させるためには、この調和をあえて乱す「初期位相オフセット」や「環境の非対称性」が必要である。
