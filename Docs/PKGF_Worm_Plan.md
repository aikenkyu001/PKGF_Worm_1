- **意味**: 多様体内の「意志（流動）」が Action セクターを通じて物理的な力へ相転移する。

### 2.2 感覚入力：計量共鳴 (Environment-to-Metric)

### 2.3 固有受容感覚：内的緊張 $A$ への還流 (Proprioception-to-Tension)

- **実装**: $A_{new} = A_{old} + \alpha \cdot \text{BodyCurvature}$
- **意味**: 身体の状態が多様体上の「摩擦」として感じられることで、自己参照的な閉ループ（意識の萌芽）を形成する。
